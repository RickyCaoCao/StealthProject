# StealthProject

Based on Unity's Tutorial

Modifications:
-Health bar
-1st and 3rd Person Camera (compared to over top)

Controls:
1. WASD to move
2. C/L to switch to 3rd person camera
3. Space for 1st person camera
4. X to shout (not very useful)
